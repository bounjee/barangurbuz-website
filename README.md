# Baran Gürbüz – Private Mathematics Tutor Website

This project is a modern and responsive promotional website created for Mathematics Teacher **Baran Gürbüz**. The site aims to present his teaching philosophy, professional experience, private tutoring services, and contact information in a clear and trustworthy way. The layout focuses on simplicity, readability, and smooth user experience across all devices.

## Features

The website includes sections for introducing the tutor, explaining the teaching approach, and providing service details for students and parents. It also offers a clean contact area to make communication easier. The design highlights professionalism and aims to build confidence for potential students.

## Technologies Used

The project is built using standard web development technologies such as HTML, CSS, and JavaScript. The structure is lightweight, easy to maintain, and optimized for fast loading times.
